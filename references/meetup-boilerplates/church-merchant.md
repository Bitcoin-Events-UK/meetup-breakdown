## Church / Merchant Event Listings

[**Organisation Name**](https://link.to/organisation): Brief description. 📍 Full venue address 🕖 Time on Day Date.

---

**Pattern:**
- No standard emoji — use context (⛪ for church, 🍽️ for restaurant, 🏪 for shop)
- Organisation or venue name + link
- One-line description of the event format
- Full venue address (including postcode)
- Start and end time
- "Bitcoin accepted" only if the venue is the merchant being featured

**Notes:**
- Church events often have a more relaxed format — talks, Q&A, or informal catching up
- If it's a one-off merchant event, flag it in the draft and note the date clearly
- Merchant events listed here are Bitcoin Events UKorganised or supported events — not general paid advertising
