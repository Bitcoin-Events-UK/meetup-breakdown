## Pub Meetup Listings

🍺[**Meetup Name**](https://link.to/meetup): One-line description. 📍 Full venue address 🕖 Time on Day Date.

---

**Pattern:**
- Emoji: 🍺
- Meetup name + link (Twitter/X, Meetup, website, or Telegram)
- One compelling line — what makes this meetup worth attending
- Venue name and full address
- Time and day
- Note "Bitcoin accepted" if verified

**Notes:**
- Most pub meetups run 18:00–20:00 or 19:00–21:00
- If the venue changes month-to-month, flag in the draft
- Always verify "bitcoin accepted" claim — don't repeat if unverified
- If there's no website, use the Meetup.com or Telegram link
