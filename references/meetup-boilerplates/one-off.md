## One-Off / Special Event Listings

🎯[**Event Name**](https://link.to/event): Description of what the event is and why it's worth attending. 📍 Full venue address 🕖 Time on Day Date.

---

**Pattern:**
- Emoji: 🎯
- Event name + link (Eventbrite, website, etc.)
- Description — what it is, format, who it's for
- Full venue address
- Time, day, and date
- Any ticket or booking requirements

**Notes:**
- One-off events get a 🎯 to distinguish from regular meetups
- Always flag if booking required — don't let people turn up to a full event
- If it's a recurring event that moved or a special edition, note that in the description
